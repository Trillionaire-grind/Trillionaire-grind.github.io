🎣 How to Build a Fishing App

If you're a fishing enthusiast or know someone who is, you’ve probably wished for a simple way to log catches, spots, and weather conditions. In this guide, we’ll create a basic fishing log web app where users can record catches and view their personal fishing history.

🎯 What Are We Building?

A simple fishing log app where users can:

  • Log new catches (species, location, weight, notes)
  • View all previous catches

We'll use Node.js, Express, HTML, CSS, and JavaScript. For simplicity, we'll store catch data in memory.

🧰 Tools You’ll Need

  • Node.js (download)
  • Express.js for backend
  • HTML/CSS/JavaScript for the frontend

📁 Step 1: Set Up Your Project

mkdir fishing-app
cd fishing-app
npm init -y
npm install express

🔧 Step 2: Create the Backend

Create a file called server.js:

// server.js
const express = require('express');
const app = express();
const PORT = 3000;

let catches = [];

app.use(express.static('public'));
app.use(express.json());

app.get('/api/catches', (req, res) => {
  res.json(catches);
});

app.post('/api/catches', (req, res) => {
  const { species, location, weight, notes } = req.body;
  if (species && location) {
    catches.push({ species, location, weight, notes, time: new Date() });
    res.status(201).json({ message: 'Catch logged' });
  } else {
    res.status(400).json({ error: 'Species and location are required' });
  }
});

app.listen(PORT, () => {
  console.log(`Fishing app running at http://localhost:${PORT}`);
});

3. app.js

async function loadCatches() {
  const res = await fetch('/api/catches');
  const data = await res.json();
  const container = document.getElementById('catches');
  container.innerHTML = '';

  data.forEach(catchData => {
    const div = document.createElement('div');
    div.className = 'catch';
    div.innerHTML = `
      <strong>${catchData.species}</strong> at ${catchData.location}<br>
      Weight: ${catchData.weight || 'N/A'} kg<br>
      Notes: ${catchData.notes || 'None'}<br>
      Time: ${new Date(catchData.time).toLocaleString()}
    `;
    container.appendChild(div);
  });
}

document.getElementById('catch-form').addEventListener('submit', async (e) => {
  e.preventDefault();
  const species = document.getElementById('species').value;
  const location = document.getElementById('location').value;
  const weight = document.getElementById('weight').value;
  const notes = document.getElementById('notes').value;

  await fetch('/api/catches', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ species, location, weight, notes }),
  });

  e.target.reset();
  loadCatches();
});

loadCatches();
