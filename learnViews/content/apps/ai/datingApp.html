❤️ How to Build a Dating App: A Beginner’s Guide

Dating apps have revolutionized how people meet, connect, and form relationships. If you’ve ever wondered what it takes to build one yourself, this guide will walk you through the basic steps of creating a simple dating app using modern web technologies.

We’ll cover both the frontend and backend, giving you a solid foundation to expand into a full-featured app later.

💡 What Are We Building?

We’ll build a simple dating web app that lets users:

  • Create a basic profile
  • Browse other users
  • "Like" profiles they’re interested in

We’ll use in-memory storage for now, just to understand the flow. No logins or databases yet — baby steps!

🧰 Tools You’ll Need

  • Node.js (download)
  • Express.js for the backend API
  • HTML/CSS/JavaScript for the frontend
  • A code editor (like VS Code)

📁 Step 1: Set Up Your Project

mkdir dating-app
cd dating-app
npm init -y
npm install express

🔧 Step 2: Create the Backend

Create a server.js file in your root directory:

// server.js
const express = require('express');
const app = express();
const PORT = 3000;

let users = [];
let likes = {};

app.use(express.static('public'));
app.use(express.json());

// Get all users
app.get('/api/users', (req, res) => {
  res.json(users);
});

// Add a new user
app.post('/api/users', (req, res) => {
  const { name, bio } = req.body;
  if (name && bio) {
    const id = users.length + 1;
    users.push({ id, name, bio });
    res.status(201).json({ message: 'User added' });
  } else {
    res.status(400).json({ error: 'Name and bio required' });
  }
});

// Like a user
app.post('/api/like', (req, res) => {
  const { fromId, toId } = req.body;
  if (!likes[fromId]) likes[fromId] = [];
  likes[fromId].push(toId);
  res.json({ message: 'Like recorded' });
});

app.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}`);
});

3. app.js

async function loadUsers() {
  const res = await fetch('/api/users');
  const users = await res.json();

  const container = document.getElementById('user-list');
  container.innerHTML = '';

  users.forEach(user => {
    const div = document.createElement('div');
    div.className = 'user';
    div.innerHTML = `<strong>${user.name}</strong><br>${user.bio}<br>
      <button onclick="likeUser(${user.id})">Like</button>`;
    container.appendChild(div);
  });
}

document.getElementById('new-user-form').addEventListener('submit', async (e) => {
  e.preventDefault();
  const name = document.getElementById('name').value;
  const bio = document.getElementById('bio').value;

  await fetch('/api/users', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ name, bio }),
  });

  loadUsers();
});

async function likeUser(toId) {
  await fetch('/api/like', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ fromId: 1, toId }), // hardcoded user for demo
  });
  alert('You liked this user!');
}

loadUsers();
